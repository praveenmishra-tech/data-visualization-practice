# Data Visualization Practice 

This repository contains my practice work on 3 major Python data visualization libraries.

##  Folder Structure

- **matplotlib/** – Basic to advanced static plots (line, bar, scatter, subplots)
- **seaborn/** – Statistical visualizations (heatmaps, distribution plots)
- **plotly/** – Interactive charts and dashboards

##  Tech Stack

- Python
- Matplotlib
- Seaborn
- Plotly

##  How to Run

pip install matplotlib seaborn plotly

## 📌 About

This repo showcases my learning journey in data visualization — from beginner-level static plots to interactive dashboards.
